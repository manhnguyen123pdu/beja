PHẦN 1: VÒNG ĐỜI SERVLET

1.1. Khái niệm

Vòng đời Servlet là quá trình từ khi Servlet được tạo ra, xử lý request, đến khi bị hủy bỏ. Toàn bộ quá trình này được quản lý bởi Servlet Container (Tomcat).

1.2. Các giai đoạn

┌─────────────────┐ │ LOADING │ ← Container nạp class Servlet └─────────────────┘ ↓ ┌─────────────────┐ │ INSTANTIATION │ ← Tạo instance (new) └─────────────────┘ ↓ ┌─────────────────┐ │ INITIALIZATION │ ← Gọi init() - 1 lần └─────────────────┘ ↓ ┌─────────────────┐ │ REQUEST HANDLING│ ← Gọi service() - nhiều lần └─────────────────┘ ↓ ┌─────────────────┐ │ DESTRUCTION │ ← Gọi destroy() - 1 lần └─────────────────┘

1.3. Ba phương thức chính

init()

public void init() throws ServletException // hoặc public void init(ServletConfig config) throws ServletException

Được gọi 1 lần duy nhất khi Servlet khởi tạo. Dùng để kết nối DB, đọc cấu hình.

service()

public void service(ServletRequest req, ServletResponse res) throws ServletException, IOException

Được gọi cho mỗi request. Quyết định gọi doGet/doPost.

destroy()

public void destroy()

Được gọi 1 lần khi Servlet bị hủy. Dùng để đóng kết nối, giải phóng tài nguyên.

📌 Ví dụ 1: Minh họa vòng đời

LifecycleServlet.java
package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; @WebServlet("/lifecycle") public class LifecycleServlet extends HttpServlet { @Override public void init() throws ServletException { System.out.println("=== 1. init() ==="); System.out.println("Servlet được khởi tạo"); } @Override protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { System.out.println("=== 2. service() ==="); System.out.println("Request: " + req.getMethod() + " " + req.getRequestURI()); super.service(req, resp); } @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { System.out.println("=== 3. doGet() ==="); resp.setContentType("text/html;charset=UTF-8"); var out = resp.getWriter(); out.println("<html><body>"); out.println("<h2>Vòng đời Servlet</h2>"); out.println("<p>Xem console để thấy log</p>"); out.println("</body></html>"); } @Override public void destroy() { System.out.println("=== 4. destroy() ==="); System.out.println("Servlet bị hủy"); } }

PHẦN 3: HTTPSESSION - QUẢN LÝ PHIÊN LÀM VIỆC

3.1. Khái niệm Session

Session là vùng lưu trữ dữ liệu riêng cho từng người dùng, tồn tại xuyên suốt nhiều request.

Đặc điểm:

  • Mỗi user có một session riêng
  • Dữ liệu lưu ở server
  • Tự động hết hạn sau thời gian nhất định
  • Định danh bằng Session ID (lưu trong cookie)

3.2. Các phương thức chính

Lấy session:

HttpSession session = request.getSession(); // tạo mới nếu chưa có HttpSession session = request.getSession(true); // giống trên HttpSession session = request.getSession(false); // null nếu chưa có

Thao tác với dữ liệu:

session.setAttribute("username", "admin"); // lưu dữ liệu String user = (String) session.getAttribute("username"); // đọc dữ liệu session.removeAttribute("username"); // xóa một dữ liệu session.invalidate(); // hủy toàn bộ session

Thông tin session:

String id = session.getId(); // lấy ID long time = session.getCreationTime(); // thời gian tạo long last = session.getLastAccessedTime(); // lần cuối truy cập session.setMaxInactiveInterval(30*60); // set thời gian sống (giây) int interval = session.getMaxInactiveInterval(); // lấy thời gian sống

📌 Ví dụ 3: Đếm số lần truy cập bằng Session

SessionCounterServlet.java
package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; import javax.servlet.http.HttpSession; @WebServlet("/session-counter") public class SessionCounterServlet extends HttpServlet { @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { HttpSession session = req.getSession(); Integer count = (Integer) session.getAttribute("count"); if (count == null) { count = 1; } else { count++; } session.setAttribute("count", count); resp.setContentType("text/html;charset=UTF-8"); var out = resp.getWriter(); out.println("<html><body>"); out.println("<h2>Đếm bằng Session</h2>"); out.println("<p>Session ID: " + session.getId() + "</p>"); out.println("<p>Bạn đã truy cập: " + count + " lần</p>"); out.println("</body></html>"); } }

PHẦN 4: COOKIE

4.1. Khái niệm Cookie

Cookie là dữ liệu nhỏ được server gửi xuống và lưu ở trình duyệt, được gửi lại server mỗi request.

Đặc điểm:

  • Lưu ở client (trình duyệt)
  • Dung lượng tối đa 4KB
  • Có thời gian sống
  • Kém an toàn hơn session

4.2. Các phương thức

Tạo cookie:

Cookie cookie = new Cookie("username", "admin"); cookie.setMaxAge(60*60*24); // 1 ngày cookie.setPath("/"); response.addCookie(cookie);

Đọc cookie:

Cookie[] cookies = request.getCookies(); if (cookies != null) { for (Cookie c : cookies) { if (c.getName().equals("username")) { String value = c.getValue(); } } }

Xóa cookie:

Cookie cookie = new Cookie("username", ""); cookie.setMaxAge(0); cookie.setPath("/"); response.addCookie(cookie);

📌 Ví dụ 4: Tạo và đọc Cookie

CookieDemoServlet.java
package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.Cookie; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; @WebServlet("/cookie-demo") public class CookieDemoServlet extends HttpServlet { @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { // Tạo cookie mới Cookie visitCookie = new Cookie("lastVisit", new java.util.Date().toString()); visitCookie.setMaxAge(30*24*60*60); // 30 ngày visitCookie.setPath("/"); resp.addCookie(visitCookie); resp.setContentType("text/html;charset=UTF-8"); var out = resp.getWriter(); out.println("<html><body>"); out.println("<h2>Cookie Demo</h2>"); // Đọc tất cả cookie Cookie[] cookies = req.getCookies(); if (cookies != null) { out.println("<h3>Các cookie hiện có:</h3>"); out.println("<ul>"); for (Cookie c : cookies) { out.println("<li>" + c.getName() + " = " + c.getValue() + " (sống: " + c.getMaxAge() + "s)</li>"); } out.println("</ul>"); } else { out.println("<p>Chưa có cookie nào</p>"); } out.println("</body></html>"); } }

package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; import javax.servlet.http.HttpSession; @WebServlet("/login") public class LoginServlet extends HttpServlet { @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { HttpSession session = req.getSession(false); if (session != null && session.getAttribute("username") != null) { resp.sendRedirect("welcome"); } else { req.getRequestDispatcher("login.jsp").forward(req, resp); } } @Override protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { String username = req.getParameter("username"); String password = req.getParameter("password"); if ("admin".equals(username) && "123".equals(password)) { HttpSession session = req.getSession(); session.setAttribute("username", username); session.setAttribute("loginTime", new java.util.Date().toString()); resp.sendRedirect("welcome"); } else { req.setAttribute("error", "Sai tên đăng nhập hoặc mật khẩu!"); req.getRequestDispatcher("login.jsp").forward(req, resp); } } }

package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.Cookie; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; import javax.servlet.http.HttpSession; @WebServlet("/login-remember") public class LoginRememberServlet extends HttpServlet { @Override protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { String username = req.getParameter("username"); String password = req.getParameter("password"); String remember = req.getParameter("remember"); if ("admin".equals(username) && "123".equals(password)) { HttpSession session = req.getSession(); session.setAttribute("username", username); if (remember != null) { Cookie cookie = new Cookie("rememberUsername", username); cookie.setMaxAge(30 * 24 * 60 * 60); // 30 ngày cookie.setPath("/"); resp.addCookie(cookie); } else { Cookie cookie = new Cookie("rememberUsername", ""); cookie.setMaxAge(0); cookie.setPath("/"); resp.addCookie(cookie); } resp.sendRedirect("welcome"); } else { req.setAttribute("error", "Sai tên đăng nhập hoặc mật khẩu!"); req.getRequestDispatcher("login-remember.jsp").forward(req, resp); } } }

📌 Bài tập 2: Lưu theme bằng Cookie

package test; import java.io.IOException; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.Cookie; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; @WebServlet("/theme") public class ThemeServlet extends HttpServlet { @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { String theme = req.getParameter("theme"); if (theme != null) { Cookie cookie = new Cookie("theme", theme); cookie.setMaxAge(30 * 24 * 60 * 60); // 30 ngày cookie.setPath("/"); resp.addCookie(cookie); } // Đọc cookie hiện tại String currentTheme = "light"; Cookie[] cookies = req.getCookies(); if (cookies != null) { for (Cookie c : cookies) { if ("theme".equals(c.getName())) { currentTheme = c.getValue(); break; } } } resp.setContentType("text/html;charset=UTF-8"); var out = resp.getWriter(); String bgColor = "dark".equals(currentTheme) ? "#333" : "white"; String textColor = "dark".equals(currentTheme) ? "white" : "black"; out.println("<html><body style='background:" + bgColor + ";color:" + textColor + "'>"); out.println("<h2>Chọn theme</h2>"); out.println("<a href='theme?theme=light'>Light</a> | "); out.println("<a href='theme?theme=dark'>Dark</a>"); out.println("<p>Theme hiện tại: " + currentTheme + "</p>"); out.println("</body></html>"); } }

📌 Bài tập 3: Giỏ hàng đơn giản

Thêm sản phẩm vào giỏ hàng và lưu trong Session.

CartServlet.java
package test; import java.io.IOException; import java.util.ArrayList; import java.util.List; import javax.servlet.ServletException; import javax.servlet.annotation.WebServlet; import javax.servlet.http.HttpServlet; import javax.servlet.http.HttpServletRequest; import javax.servlet.http.HttpServletResponse; import javax.servlet.http.HttpSession; @WebServlet("/cart") public class CartServlet extends HttpServlet { @Override protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException { HttpSession session = req.getSession(); List<String> cart = (List<String>) session.getAttribute("cart"); if (cart == null) { cart = new ArrayList<>(); session.setAttribute("cart", cart); } String action = req.getParameter("action"); String product = req.getParameter("product"); if ("add".equals(action) && product != null) { cart.add(product); } else if ("remove".equals(action) && product != null) { cart.remove(product); } resp.setContentType("text/html;charset=UTF-8"); var out = resp.getWriter(); out.println("<html><body>"); out.println("<h2>Giỏ hàng</h2>"); out.println("<h3>Sản phẩm:</h3>"); out.println("<ul>"); out.println("<li><a href='cart?action=add&product=Laptop'>Thêm Laptop</a></li>"); out.println("<li><a href='cart?action=add&product=Phone'>Thêm Phone</a></li>"); out.println("<li><a href='cart?action=add&product=Tablet'>Thêm Tablet</a></li>"); out.println("</ul>"); out.println("<h3>Giỏ hàng hiện tại:</h3>"); out.println("<ul>"); for (String item : cart) { out.println("<li>" + item + " <a href='cart?action=remove&product=" + item + "'>[Xóa]</a></li>"); } out.println("</ul>"); out.println("</body></html>"); } }

TỔNG KẾT BUỔI 3

Lưu ý quan trọng:

  • Servlet là Singleton - chỉ có 1 instance cho tất cả request

  • Cookie lưu ở client - phù hợp cho preferences
  • Luôn kiểm tra session trước khi cho phép truy cập trang bảo mật
